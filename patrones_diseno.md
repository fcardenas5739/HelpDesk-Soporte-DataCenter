# Patrones de diseño aplicados al sistema Help Desk

## Singleton
Se utiliza para controlar una única instancia de conexión o configuración general del sistema.

## Factory Method
Se aplica para crear diferentes tipos de incidentes, como fallos de hardware, red o software.

## Observer
Se utiliza para notificar automáticamente a los técnicos cuando se registra o actualiza un ticket.

## MVC
Se plantea como estructura general para separar la vista, la lógica de control y el modelo de datos.
